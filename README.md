## Getting Started

Run the server in dev mode (Requires NodeJS):

1. Clone the Repository

```
$ git clone https://github.com/DavidRR-F/my-portfolio.git
```

2. Install Dependencies

```
$ npm install
```

3. Run Dev Enviroment

```
$ npm run dev
```
